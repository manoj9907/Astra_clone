# Install bun

<https://bun.sh/docs/installation>

# Install npm modules

```bash
bun install
```

# Run development server

```bash
bun run dev
```

# Linting and formatting

```bash
# Automatically formats all files using Prettier
bun fmt

# Runs eslint recursively on all files in the current subdirectory
bun lint
```
